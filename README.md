## 🚀 Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/martinech/Olympic-games-app.git
   cd Olympic-games-app
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the application:
   ```sh
   npm start
   ```
